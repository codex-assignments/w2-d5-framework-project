# w2-d5-framework-project
Objective Interview on building a small, responsive one-page site with a styling framework.

Requirements
Live session with a TA or instructor.
Show your deployed site (GitHub Pages) and the public repo w2-d5-framework-project.
Be ready to make one small live change, such as:
add a responsive section using grid or flex
adjust spacing or type scale with framework utilities
add or fix a focus style for an interactive element
Demonstrate responsive behavior at small, medium, and large breakpoints.
Answer brief questions about the utilities and components you used.
